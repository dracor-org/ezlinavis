## Development

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

### Synopsis

```bash
git clone https://github.com/dracor-org/ezlinavis.git
cd ezlinavis
pnpm run install
pnpm run test
pnpm run start
```
